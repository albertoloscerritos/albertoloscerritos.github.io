Que es JavaScript

JavaScript Es un lejuage de progamacion, no de etiquetas. Esto quiere decir que nos ayuda a ponerle funcionalidad a nuestra pagina. Sin embargo tiene algunas restricciones, pero con la unica finalidad de protejer al usuario de paginas maliciosas. Y no se confundan. Que no tiene nada que ver con el famoso lenguaje de programacion Java, son dos cosas diferentes.

Que es Jquery

Jquery es una herramienta muy util para utlizar JavaScript. Se puede decir que es como una libreria de dicho leguanje. Y es de vital ayuda para el programador en especial cuando este no tiene un amplio conocimiento del lenguaje
